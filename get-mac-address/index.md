# 获取设备的MAC地址


这篇文章提供了获取接入华为交换机的每个设备的MAC地址的方法。<!--more-->

## 获取MAC地址

通过串口进入交换机命令行

```bash
#进入系统视图
system
#显示每个端口的mac地址
display mac address
```
